# Arduino Push Button

A beginner Arduino project demonstrating how to read digital input from a push button and control an LED.

## Features

* Reads a push button using `digitalRead()`
* Turns an LED on and off based on button state
* Demonstrates basic digital input and output

## Components

* Arduino Uno
* Push Button
* LED
* 220Ω resistor
* Breadboard
* Jumper wires

## Skills Learned

* `pinMode()`
* `digitalRead()`
* `digitalWrite()`
* Basic circuit wiring
* Digital input handling

## Future Improvements

* Add software debouncing
* Toggle LED with one button press
* Control multiple LEDs
